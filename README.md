# jorian-web
开箱即用的后台管理框架配套的前端框架，基于Vue + axios + webpakage 开发
后台框架地址：https://github.com/Jorian93/jorian-framework.git

1.支持根据配置或后台返回的json数据动态渲染菜单;  
2.支持SVG图标快捷引入;    
3.使用axios封装请求  
3.支持mock数据便于前后端分离开发;  
4.集成了opnlayers地图引擎;  
5.集成富文本插件tinymce;  
6.及更多组件
