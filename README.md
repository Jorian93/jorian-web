# jorian-web
基于Vue的后台管理前端框架，支持根据后台返回的菜单json数据进行页面渲染，支持SVG图标快捷引入，支持mock数据便于前后端分离开发，集成了opnlayers地图引擎，更有富文本tinymce等更多组件
